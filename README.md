---Automated Library Inventory & Waitlist System---

Overview:  
This is a Java-based Object-Oriented application designed to solve real-world problems in libraries. Using OOP, the process of borrowing and waitlisting customers can be automated.

Features:  
-Automated Inventory Management  
-Automated Waitlist System  
-Notification System
